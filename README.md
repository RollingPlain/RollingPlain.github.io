# Personal Homepage Template

This is a clean academic personal homepage template for GitHub Pages.

## How to use

1. Replace the placeholder text in `index.html`.
2. Replace `assets/avatar.svg` with your own `avatar.jpg` or `avatar.png`.
3. Update the image path in `index.html` if the file name changes.
4. Push these files to a GitHub repository named `your-github-username.github.io`.
5. Enable GitHub Pages in repository settings if it is not already enabled.

## Main sections

- About Me
- News
- Selected Publications
- Experience
- Footer
